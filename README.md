# todo-app

> A simple Todo App

## Build Setup

``` bash
# install dependencies
npm install
npm install vue-router@3.0.1
npm install vuex@3.0.1

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
